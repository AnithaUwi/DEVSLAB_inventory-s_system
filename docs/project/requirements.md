
# What to Build

## Must Have (Week 1)
1. **Login System**
   - Users can login
   - Different roles work differently

2. **Item Management**
   - Add/edit items (BRAKE-001, ENGINE-002)
   - Set reorder levels (buy more when < 5)

3. **Stock Operations**
   - Receive new stock
   - Transfer between branches
   - Issue to mechanics
   - See stock levels

## Should Have (Week 2)
4. **Reports**
   - Stock list per branch
   - Movement history
   - Export to Excel

5. **Dashboard**
   - See quick numbers
   - Low stock alerts
