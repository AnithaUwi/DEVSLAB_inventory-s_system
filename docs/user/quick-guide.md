# Quick User Guide

## For Storekeepers:
1. **Login** with your email/password
2. **Receive Stock** (when new items arrive)
3. **Issue Parts** (when mechanic needs)
4. **Do Daily Count** (check physical stock)

## For Managers:
1. **Approve** big transfers
2. **View Reports** (stock levels, usage)
3. **Reorder** when stock is low

## Common Tasks:
### Add New Item:
1. Go to Items → Add New
2. Enter: Code, Name, Category
3. Set minimum stock (like 5)
4. Save

### Transfer Items:
1. Go to Stock → Transfer
2. Select "From" branch and "To" branch
3. Choose items and quantities
4. Submit
